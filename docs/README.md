# Documentation: AI Compliance-by-Design Framework

This directory contains the complete documentation for the Healthcare AI Compliance-by-Design Framework. The documents are organized to guide system architects, analysts, and integrators through the key principles, architecture, and governance mechanisms required for AI-driven healthcare systems in regulated environments.

## 🧭 How to Use This Documentation

1. **Start with** `00-overview.md` to understand the framework's goals and audience.
2. **Review** `01-scope-and-non-scope.md` to confirm applicability to your project.
3. **Study the core principles** in `02-core-principles.md` — these inform all downstream decisions.
4. **Explore architecture and governance** sections (`03` through `09`) based on your current focus area.
5. **Use** `10-validation-and-acceptance.md` and `11-deployment-patterns.md` when preparing for implementation.
